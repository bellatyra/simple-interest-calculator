## How It Works

1. The program prompts the user to enter:

   - Time period (in years)

3. It displays the simple interest and the total amount (principal + interest).

## Requirements

- Python 3.x

## How to Run

1. Clone or download this repository.
2. Open a terminal in the project folder.
3. Run the script:
      python interest_calculator.py
4. Enter the requested values when prompted.
## Example
      Enter principal amount: 1000
